# Spacerad.io

Personal blog based on Astro. Still working in progress, documents will come later.

## Roadmap

- Blog Skeleton
- Theme, made bokushi origin to spaceraid.io styles
- Telegram Channel
- Consistency
- Deploy
- Migrate comment system to Remark42

### Stack
1. Astro and Vue
2. UnoCSS + SCSS
3. Cloudflare Pages
4. MDX
5. Comment System - Remark42

### Feats
1. Responsive design
2. Dark mode
3. Telegram channel integration

### Dev Tools
1. Biome
2. Husky
